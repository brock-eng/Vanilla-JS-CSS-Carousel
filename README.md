# Vanilla-JS-CSS-Carousel
Generalized Vanilla-JS-CSS Carousel for use with an arbitrary number of panels and content.

                                    ## User HTML Panel
                                </li>
                                <li class="carousel_slide">
                                    ## User HTML Panel
                                </li>
                                <li class="carousel_slide">
                                    ## User HTML Panel
                                </li>
                                <li class="carousel_slide">

                        </div>     
                        <button class="carousel_button carousel_button--right">
